IDEA使用技巧（持续更新）

|字数总计:508|阅读时长:1分钟|阅读量:

一、IDEA的使用

1、快捷键

组合键/字母 作用
main public static void main(String[] args)
sout System.out.println();
/**回车 自动生成文档注释
Ctrl+/ 自动生成单号注释
Ctrl+shift+/ 自动生成多行注释
Ctrl+Alt+L 自动对齐代码（行数，括号等等）
Alt+Ins 快速生成代码块（构造方法，set，get等等）
Ctrl+n 快速查找类，或者所有。
Ctrl+单击 查看源代码
Alt+Ins 重写toString和equals方法
Ctrl+D 复制这一行内容
Ctrl+x 删除这一行内容
Alt+方向键 切换所显示的文件
Ctrl+Alt+t 调出if,for,try.catch.finally等代码
fori 快速生成for int i循环
对象后加.var 自动生成对象，补全前面的类名
Alt+7 可以查看类的字段、属性、方法，是否继承等
Ctrl+H 查看当前类的继承关系
Ctrl+Tab 切换导航栏页面，比alt+方向键更强
Ctrl+w 选中一个或者多个单词
shift+enter 创建新的下一行
Ctrl+Alt+Enter 创建新的上一行
Alt+T A8翻译插件，快速翻译idea

2、光标快捷键

快捷键 作用
ctrl+w 选择一个或者多个单词
ctrl +left（方向键） 向左移动一个单词
crtl+shift+M 移动到大括号，切换
crtl+[/] 移动至代码块开始/结束
alt +up（方向键） 上一个方法
crtl+shift+left 向左选中一个单词
alt+shift+单击 批量在单击位置添加代码

二、插件

1、A8Translate

翻译插件，使用快捷键Alt+T可以快速翻译

2、Markdown Editor

查看，编辑markdown文件

3、Nyan Progress Bar

右下角进度条变成一只拖着彩虹的猫

4、Rainbow Brackets

彩虹括号神器，代码括号颜色自动生成，再也不用担心看不起括号了

5、Lom book

快捷生成类的构造函数，set&get方法

6、Easy Code

快速对数据库表进行逆向工程，生成实体类，映射类，实现类，控制类等等
